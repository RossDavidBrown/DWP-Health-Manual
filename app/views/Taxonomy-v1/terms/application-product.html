Application product

A group of technical components that combine to support the delivery of a service or services.

Service application products

For example:

  • Service Signposting
  • Service Citizen Attributes
  • Service Payments

Whilst the strategic aim is to consilidate similar application products into as few implementations as possible it is recognised that services will need to tailor individual application producs to deliver specific legislative and policy driven user journeys.

Enterprise application products

For example:

  • DWP Appointments
  • DWP Identity
  • DWP Accounting

There are a core set of business capabilities that have been identified as candidates to be udnerpinned by a consolidated suite of reusable, easily consumaple enterprise application products.

3rd party application products

For example:

  • GOV.UK Notify
  • HMRC Earnings
  • banking system providers

There are a number of application products that DWP wouldn't propose to build, customise or configure for consumption. These application products will typlically be 'software as a service' (SaaS) offerings that we will procure and expose as enterprise offerings via the DWP API gateway.
